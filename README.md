# VerifyBusiness

مشروع `VerifyBusiness` مبني باستخدام `Next.js` ويدعم تعدد اللغات.

## التشغيل المحلي

```bash
npm install
npm run dev
```

افتح:

```text
http://localhost:3000
```

إذا كان المنفذ مستخدماً:

```bash
npm run dev -- --port 3001
```

## التحقق قبل النشر

```bash
npm run lint
npm run build
```
